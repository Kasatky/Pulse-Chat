Страница с единым полем для общего чата

<h3>Release 0.1</h3>
Деплой минимально работающего приложения

<h3>Release 1</h3>
Registration/Login фронт и бэк, добавление никнейма(он может быть не уникальным, для идентификации используем циферный юсер айди из базы, email должен быть уникальным, используем проверку при регистрации)

<h3>Release 2</h3>
Чаты пользователя с пользователем, поиск человека по нику, возможность написать

<h3>Release 3</h3>
Личный кабинет с возможностью менять никнейм, добавлением аватара(опционально)

<h3>Release 4</h3>
Окончательный дизайн под продакшен

<h3>Release 5</h3>
Групповые чаты с большим количеством людей

<h3>Release 6</h3>
Возможность отправки фото и файлов в чате
